README: INTRODUCTION

'Xfce4_Session_Automator' by Ludovic Lemoine
______________________________________________________________________________________________________________________________________________

Welcome to the Xfce4-Session-Automator documentation.
Please be aware that this software is still under ongoing development.
The software licensing is displayed at the document's end.

Log files:
C:\Program Files\Xfce4_Session_Automator\transcripts

First Run Instructions:
Start the 'Distribution Selector' Desktop shortcut. 
or
Start Menu > Xfce4_Session_Automator > Distribution Selector
or
C:\Program Files\Xfce4_Session_Automator\launchers\Distribution Selector.lnk

----------------------------------------------------------------------------------------------------------------------------------------------

Uninstall the software:
Start Menu > Xfce4_Session_Automator > Uninstall
or
C:\Program Files\Xfce4_Session_Automator\uninst000.exe

----------------------------------------------------------------------------------------------------------------------------------------------

You might need to accept temporary PowerShell Execution Policy change.

README File: C:\Program Files\Xfce4_Session_Automator\README.txt

SYSTEM & SOFTWARE REQUIREMENTS:
______________________________________________________________________________________________________________________________________________

Minimum OS Level: Microsoft Windows 10 Home Edition (1803) (Windows 10 IoT, Windows 10 S & Windows 10 N must to be tested / not supported yet)

Install the latest minor Microsoft Windows Updates and reboot your computer before beginning this procedure.
Changing the installation path is not recommended.
